# Perceptron Trick Using Sigmoid

-> This project demonstrates the **Perceptron Trick combined with the Sigmoid activation function**, bridging the gap between classical perceptrons and logistic regression.  
-> Instead of using a step function, we apply a smooth sigmoid output and update weights using gradient-based corrections.

## Contents:-
- Perceptron trick refresher
- Sigmoid activation function
- Weight updates using sigmoid gradient
- Numerical implementation with NumPy
- Decision boundary visualization

## Key Concepts:-
- Difference between perceptron update & sigmoid-based update  
- Smoothing the decision boundary  
- Logistic-style learning with perceptron intuition

## Libraries Used:-
- NumPy  
- Matplotlib  
- Scikit-learn (optional)

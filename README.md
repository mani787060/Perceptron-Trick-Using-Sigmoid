# Perceptron Trick Using Sigmoid

## 📌 Project Overview

This project demonstrates the **Perceptron Trick using the Sigmoid Activation Function**, showing how the traditional perceptron learning rule evolves into a gradient-based learning approach similar to **Logistic Regression**.

Using a synthetic dataset generated with Scikit-Learn, the notebook explains how the sigmoid function produces smooth probability outputs and how model parameters are updated iteratively to improve classification performance.

---

## 🎯 Objectives

* Understand the Perceptron Trick with Sigmoid Activation
* Learn how the Sigmoid function works
* Study gradient-based weight updates
* Visualize the decision boundary
* Understand the transition from Perceptron to Logistic Regression

---

## 📂 Dataset

**Dataset Used:** `make_classification`

A synthetic binary classification dataset generated using Scikit-Learn to demonstrate sigmoid-based learning and decision boundary optimization.
